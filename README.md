# Overview
code to determine panstarrs color terms for calculating the AB photometric zeropoint in various filters.

# Dependencies
* speclite

# Spectral Libraries

[MILES Libray](https://research.iac.es/proyecto/miles/pages/stellar-libraries/miles-library.php) of 1000 stars.  Spectra were taken with the 2.5m INT with  2.5A resolution (Sánchez-Blázquez et al. 2006, Falcón-Barroso et al. 2011)
